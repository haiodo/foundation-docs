## API Foundation

Foundation имеет базовый API, чтобы вы могли начать работу с собственными интеграциями и расширениями. Ознакомьтесь с [документацией API](https://github.com/hcengineering/huly-examples/tree/main/platform-api#readme) для получения инструкций по настройке и примеров использования.

## Инструмент импорта Foundation

Мы предлагаем инструмент импорта, который поможет вам импортировать данные напрямую в ваше рабочее пространство Foundation.

Используя наш унифицированный формат импорта, вы можете перенести свои данные в Foundation, преобразовав их в промежуточную, человекочитаемую структуру. Ознакомьтесь с [документацией инструмента импорта Foundation](https://github.com/hcengineering/platform/tree/develop/dev/import-tool#readme) для получения инструкций по настройке и примеров.

Мы также поддерживаем импорт из [Notion](https://github.com/hcengineering/platform/blob/develop/dev/import-tool/docs/notion/README.md) и [ClickUp](https://github.com/hcengineering/platform/blob/develop/dev/import-tool/docs/clickup/README.md).
