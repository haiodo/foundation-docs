Foundation предлагает высокую степень настраиваемости через Классы. Класс — это набор или категория объектов, имеющих некоторое общее свойство. Другими словами, классы — это способ группировки объектов на основе общих свойств.

Чтобы получить доступ к настройкам классов, перейдите в настройки и нажмите `Классы` в меню `НАСТРОЙКИ РАБОЧЕГО ПРОСТРАНСТВА`. Вы увидите список всех классов в вашем рабочем пространстве. Нажатие на любой класс отобразит его свойства.

## Настройка свойств класса

В настоящее время можно настраивать только классы `Контакт` и `Задача`.

Чтобы добавить свойство, нажмите кнопку `+`. Выберите из следующих типов свойств:

- URL
- Строка — для однострочного текста
- Логическое значение — выберите Да, Нет или Н/Д
- Число
- Дата
- Ссылка — (ещё не реализовано)
- Массив — (ещё не реализовано)
- Перечисление — выберите из списка опций (см. ниже)

Вы увидите, что ваше новое свойство теперь появится на любом объекте, принадлежащем этому классу.

Чтобы удалить свойство, откройте меню настроек слева от свойства и нажмите `Удалить`. Вы также можете скрыть свойство здесь, нажав `Скрыть атрибут`.

## Перечисления

Перечисления — это тип свойства, который позволяет выбирать из списка опций. Например, вы можете добавить свойство «Как познакомились» к вашим контактам и использовать опции, такие как «Конференция», «Общий знакомый», «Холодный звонок» и т.д.

Чтобы создать перечисление, перейдите в Настройки > Перечисления и нажмите кнопку `+ Создать перечисление`. Добавьте опции и нажмите `Сохранить`.

Вы также можете создавать перечисления непосредственно из настроек Классов. После создания нового свойства с типом `перечисление` используйте кнопку `+`, чтобы создать новое перечисление для этого класса.
