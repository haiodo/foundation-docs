Модуль управления тестами Foundation позволяет вам эффективно создавать, организовывать и запускать тесты в вашем рабочем пространстве. С помощью этого модуля вы можете отслеживать статусы тестов, просматривать результаты, связывать тесты с задачами в Трекере и создавать рабочие тикеты для исполнителей.

## Настройка тестов

Нажмите кнопку `+` в левой боковой панели, чтобы начать новый тестовый проект. На данный момент тестовые проекты изолированы в модуле управления тестами; однако планируется соединить управление тестами с другими проектами рабочего пространства в ближайшем будущем.

### Создание набора тестов

Откройте выпадающее меню в верхнем левом углу и выберите `Создать набор тестов`. Укажите название и необязательное описание для вашего набора тестов.

### Создание тестового случая

В вашем наборе тестов добавьте тестовые случаи, нажав `+ Новый тестовый случай`. Введите название и необязательное описание для вашего тестового случая. Выберите статус (например, Готов, В работе) и прикрепите любые соответствующие файлы или изображения. Затем нажмите `Создать`.

### Запуск тестов

Используйте флажки рядом с каждым тестовым случаем, чтобы выбрать, какие из них включить в тестовый запуск. Затем нажмите `Запустить`.

Укажите название и описание для вашего тестового запуска. Ваш тестовый запуск может включать такие детали, как настройка окружения, инструкции по выполнению и ожидаемые результаты.
